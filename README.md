🚀 Compiler-Based Structural Code Plagiarism Detection System

A full-stack, compiler-driven plagiarism detection system that detects structural code similarity using Lexical Analysis, AST Normalization, Control Flow Graph (CFG) Comparison, Symbol Table Analysis, and Tree Edit Distance algorithms.

Unlike traditional text-based plagiarism checkers, this system identifies logical similarity even after:

Variable renaming

Formatting changes

Code reordering

Comment removal
